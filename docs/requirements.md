## 2. المتطلبات غير الوظيفية (Non-Functional Requirements)

### 2.1 الأداء
- **Time to First Byte (TTFB):** < 200ms
- **First Contentful Paint (FCP):** < 1.8s
- **Largest Contentful Paint (LCP):** < 2.5s
- **Cumulative Layout Shift (CLS):** < 0.1
- استخدام next/image للصور مع lazy loading
- Dynamic imports للمكونات الثقيلة

### 2.2 التوافقية
- **Browsers:** Chrome, Firefox, Safari, Edge (آخر نسختين)
- **Mobile:** iOS Safari, Chrome Mobile
- **Responsive:** Mobile First (320px → 1920px+)

### 2.3 الأمان
- Supabase RLS على كل الجداول
- حماية routes Admin بـ Middleware
- التحقق من الإدخالات بـ Zod
- No exposed secrets في client-side

### 2.4 SEO
- Metadata كاملة لكل صفحة
- Structured Data (JSON-LD) للمنتجات
- Sitemap.xml ديناميكي
- Robots.txt
- Canonical URLs

### 2.5 إمكانية الوصول (Accessibility)
- WCAG 2.1 AA compliance
- ARIA labels على العناصر التفاعلية
- Keyboard navigation
- Color contrast ratio > 4.5:1

### 2.6 التوطين
- دعم اللغة العربية (RTL)
- دعم اللغة الإنجليزية (LTR)
- جميع النصوص قابلة للتعديل من Admin
